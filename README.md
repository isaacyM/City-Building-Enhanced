City Building Enchanced is a mod for Age of Empires 3 

The main feature of this mod is to give you more buildings for you to build as well as technologies to research in game.

Buidlings will be added to a new settler unit that will be the only one able to build these new buildings. Which be available 
at the town center. 

Technologies added are a mixture of vanilla style additions and complex strategic decisions. Not every technology will be researhcable.
Some technologies will expand and unlock tech trees while locking out other tech trees. The reason for this is to have technologies 
be influencial in live game. Just as one chooses a deck to send cards that influences how they play the game. I want these technoligies 
to do the same and/or allow players to react to unfolding game.
